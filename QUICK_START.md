# 🚀 クイックスタートガイド

このファイルは、既にNode.jsがインストールされている方向けの簡易手順です。
詳細な手順は `SETUP_GUIDE.md` を参照してください。

---

## 1️⃣ プロジェクトフォルダに移動

```powershell
cd C:\Users\tokam\Downloads\1
```

## 2️⃣ パッケージをインストール（初回のみ）

```powershell
npm install
```

## 3️⃣ 開発サーバーを起動

```powershell
npm run dev
```

## 4️⃣ ブラウザで開く

ブラウザで以下のURLを開く：
```
http://localhost:3000
```

---

## 🛑 サーバーを停止する

PowerShellで `Ctrl + C` を押す

---

## ❓ 問題が発生した場合

詳細なトラブルシューティングは `SETUP_GUIDE.md` を参照してください。
